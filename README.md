# 사칙연산 계산기
사칙연산과 괄호의 연산자 우선 순위를 Java AWT로 구현한 계산기 

## 시작하기
### 프로그램 사용하기
> 1. 본 repository를 clone하거나 다운로드합니다.
> 2. IDE를 통해 프로젝트를 열고, `CalculatorMain.java`를 Run하여 계산기를 실행합니다.

## 개발환경
- Java 설치 버전: JDK 1.8.0_161
- IDE: Eclipse Oxygen, IntelliJ IDEA Community
